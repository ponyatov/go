## install

* https://golang.org/doc/install
* setup env variables:

```
$ setenv GOROOT = /usr/local/go
# setenv GOPATH = ~/go
```

* clone root structure

```
$ cd ~ ; git clone -o gh https://github.com/ponyatov/go
$ cd ~/go ; make packages
	go get github.com/nsf/gocode
	...
```
